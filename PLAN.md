# Plan

## Goal

Build a polished, high-converting, mobile-first website for Daniel Bower GCSE Maths Tutoring that turns parent referrals from Facebook, WhatsApp and word of mouth into booked GCSE Maths sessions.

## Content Source

The supplied PDF is the source of truth for copy and visual evidence. The site preserves the main wording from the PDF and uses page-rendered assets for:

- Daniel's portrait
- Parent feedback
- GCSE past paper tracker
- Student progress examples
- Online lesson examples

## Site Structure

- Hero
- Trust and credentials
- How GCSE Maths Support Works
- What's included
- Parent feedback
- Student progress
- Online lessons
- About Daniel
- FAQs
- Final CTA
- Footer

## Deployment Tasks

- Create or connect a GitHub repository.
- Deploy the static site to Cloudflare Pages.
- Configure `bowermaths.co.uk` as the production custom domain.
- Add or update DNS records in Cloudflare.
- Confirm HTTPS is active.

## Remaining Input Needed

- Direct WhatsApp number or `wa.me` URL.
- Any extra review screenshots or lesson screenshots that should replace the PDF-derived assets.
